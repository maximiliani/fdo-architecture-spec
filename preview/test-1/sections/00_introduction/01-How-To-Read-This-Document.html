How to Read This Document

This document is a detailed specification for FDOs. The document is primarily intended for the following audiences:

  • Anyone who wants to understand the basic ideas behind the FDO concept.
  • Software developers who want to implement the FDO concept using a specific technology.
  • Software developers who want to implement services that support the creation, manipulation and management of FDOs.
  • Software developers who want to implement clients that are able to process and interact with FDOs.

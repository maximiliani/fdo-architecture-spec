The data validation shall analyse the data acquired from [Step 9] and validate it against the various attributes found in the FDO. The validation is specific to each of the attribute types and some attributes may specify in their FDO definition, rules for verifying that the data is compliant with the value in the attribute.
Inputs
The inputs shall consist of the data from [Step 9] and the Evaluated_Profiles from [Step 8] and a list of test attributes the client wishes to evaluate the data with.
Expected behavior
For each of the test attributes, find the attribute in the list of Evaluated_Profiles and evaluate the value of the attributes and determine if the data matches.
Outputs
The output is a list of attributes whose values matched the data and a list of attributes whose values did not match the data.

This step evaluates the set of attributes within each Evaluated_Profile returned from [Step 8] to determine whether they describe an object of interest to the accessing client. The FDO Attribute Evaluation criteria are specific for each Evaluated_Profile and for each client.

Inputs
The list of Evaluated_Profile from [Step 8] and a set of client specified criteria for evaluating each of the attributes within each Profile.
Outputs
The output is true or false for each of the Validated_Profiles. Either a Profile describes an object that is relevant to the client or it is not.
Expected behavior

For each Validated_Profile, inspect the list of attributes within each of the Validated_Profile and evaluate if their values match the client’s requirements.

Clients may have different requirements such as requiring fully valid FDOs with valid FDO Profiles, verified attributes while others may have less restrictive needs.

Mandatory and Optional attributes

Two sets of attributes are defined:

[R9-1] Mandatory Attributes Set

Set of mandatory attributes that MUST be instantiated by each FDO. These are defined in

[R9-2] Optional Attributes Set

A set of harmonized optional attributes that are RECOMMENDED to be reused across use-cases as much as possible by the FDO Forum.
Each attribute is defined as its own FDO making them FAIR. There are a total of three mandatory attributes: 0.FDO/Type, 0.FDO/Profile and 0.FDO/Data. They are summarized in the root-profile 0. FDO/Root (see ). The optional attributes are summarized in table X in the appendix.

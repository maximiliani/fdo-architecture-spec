FDO records

Specify how an FDO record is realized by the given FDO implementation.

[I3-1] Map FDO record → implementation structure
Starting with the conceptional FDO record containing attributes a1,...,an, map this record to an implementation structure that contains implementation-specific attributes a'1,...,a'n.
[I3-2] Map implementation structure → FDO record
Starting with an implementation structure that contains implementation-specific attributes a'1,...,a'n, map this structure to a conceptional FDO record containing attributes a1,...,an.
